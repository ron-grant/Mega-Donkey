Mega Donkey Port Pin Assignments
Some pins used by donkey are still available for user applications, see notes.
Brackets in "Donkey Use" descriptions indicates functions that are more dedicated than others.
All functions except PB4 (Speaker) available on Expansion Header and Serial Header are available without
having to cut traces / solder in header pins.

Mega Donkey Port Pin Assignment Notes
B PC2 Drives LED Backlight via transistor T4 on LED back light equipped models.  
On EL Backlit models this pin and/or use of drive transistor is available via 
installation of header pin(s).
C PC3 Drives EL Backlight and is available on LCD header (back row) via
user installation of header pin
L  Each LED has a printed circuit jumper next to it. Cutting the jumper allows acces to the port pin, but 
LED function is lost unless assigned to another port pin.
Also, if sharing LED pin in conjunction with using LED, current flow to LED is minimal. Port pin drives LED through 
12.4K resistor connected to LED anode, then LED cathode connected to ground.
I If using Bootloader, the ISP Pins are available. If connecting an ISP programmer, then care need be 
taken to insure no contention exists between programmer and your circuitry.
M PG5 (Mega2561only)  is not available on any header.
R Pin Available on Serial Header by removing jumper and making connection with side closest to
microcontroller chip. Care needed to ensure not using handshaking for CTS/RTS lines. Also, serial port would need to 
be disabled if using RX/TX lines -- unless using for TTL levels versus RS-232 levels available on PORT0
and PORT1 DB9 connectors
S Speaker does have a Printed Circuit Jumper JP5 that can be cut if PB4 needed, losing speaker function 
unless recovered by using another port pin.
T Not Available for use on Touch Screen Models
X Soft UART Receive for Trackball (AKA UART2/UART3 receive)
Note: This pin appears on both Expansion Header and Serial Header
Z PortA (PA0…PA7) is used as the LCDs databus. In a specialized application, you can use it in conjunction with
the LCD. Of course in a very specialized application where you might opt to not use any LCD software, you would
ultimately have access to all of the port pins on the LCD header.
The pins identified on the silkscreen are those of the empty row of holes, suitable for a 20x1 header , and of the row of 
pins adjacent to the empty row which is occupied by a female connector on the reverse side of the board.
The row of pins closest to the edge of the board, has a slightly different pinout -- most notably the EL (Electro Luminescent) backlight pins
labeled HV which provides 200 volts (low current)  for EL backlight on EL backlight models.
If there is any question about EL backlight being present on your model, look for big inductor L1 next to Serial Header.
If L1 is there, you have an EL backlit model.
The function bus_in_use() is used to determine if PORTA can be taken over. See User's Guide.
